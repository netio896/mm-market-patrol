# MM Market Patrol 🇲🇲

Myanmar Market Monitoring Dashboard — real-time price tracking, alerts, and daily patrol reports.

## Features

- 📊 **Dashboard Overview** — Market summary, health score, top movers
- 🏪 **Markets** — Commodities, Currency, Consumer Goods tracking
- 🚨 **Alerts** — Price spikes, drops, supply shortages, unusual activity
- 📋 **Reports** — Daily patrol summaries with key findings

## Markets Tracked

| Category | Items |
|----------|-------|
| Commodities | Rice, Palm Oil, Gold, Onion |
| Currency | USD/MMK, THB/MMK, CNY/MMK, SGD/MMK |
| Consumer Goods | Cooking Oil, Sugar, Chicken, Eggs, Fuel, Cement, Steel |

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
